# RC Car Self-Driving — V-JEPA 2.1 Project Plan

## Overview

**Đề tài:** Action-Conditioned World Model for RC Car Navigation based on V-JEPA 2.1  
**Novel contribution:** First evaluation của V-JEPA 2 family trên mobile robot (RC car) — Meta chỉ test trên robot arm  
**Deadline:** ~2026-06-25 (1 tháng)

## Core Idea

Freeze V-JEPA 2.1 encoder (pretrained), chỉ train một **AC Predictor** nhỏ (~5M params) trên RC car video data tự collect.  
Model học: _"action nào gây ra thay đổi visual như thế nào"_  
Dùng để plan: đưa ảnh đích → tìm action sequence → thực thi trên xe thật.

## Architecture

```
[RunCam WiFiLink 2]
  --WFB-NG (wlan1, ch161, H.265)--> [PC: RTX 5070 Ti]
  ffmpeg pipe -> BGR frames (1280x720)
  V-JEPA 2.1 encoder (FROZEN) -> latent s_t (1024-dim)
  AC Predictor(s_t, a_t) -> ŝ_{t+1}
  CEM planning -> best action sequence
  [PC] --UDP 2 bytes--> [ESP32-S3 @ 192.168.1.23] -> SG90 + ESC
```

## Training Pipeline

```
frame_t   -> V-JEPA encoder (frozen) -> s_t   ─┐
action_t  ─────────────────────────────────────→ AC Predictor -> ŝ_{t+1}
frame_{t+1} -> V-JEPA encoder (frozen) -> s_{t+1} (ground truth)
Loss: MSE(ŝ_{t+1}, s_{t+1})
```

**Key optimization:** Pre-encode toàn bộ dataset offline (1 lần) → lưu latent tensors → training không cần forward qua V-JEPA nữa → nhanh 50-100x.

## Roadmap (4 tuần)

### Phase 1 — Infrastructure (tuần 1, không cần xe)
- [ ] `src/capture.py` — FrameCapture: ffmpeg pipe, queue-based, auto-reconnect
- [ ] `src/encoder.py` — VJEPAEncoder: load checkpoint, frozen, mean-pool → (1024,)
- [ ] `src/controller.py` — ESPController: UDP socket → 2 bytes [steering, throttle]
- [ ] `firmware/` — ESP32-S3: WiFi UDP server → parse 2 bytes → PWM SG90 + ESC
- [ ] Đo latency end-to-end: camera → encoder → command

### Phase 2 — Data Collection (tuần 2, cần xe)
- [ ] `src/recorder.py` — sync record video + ESP32 control log (timestamp-based)
- [ ] `src/offline_encode.py` — batch encode recorded frames → `data/latents/*.pt`
- [ ] Thu 30-60 phút demonstration data (diverse: tốc độ, góc cua, ánh sáng)

### Phase 3 — Model Training (tuần 3)
- [ ] `src/ac_predictor.py` — MLP/Transformer AC Predictor, ~5M params
- [ ] `src/train.py` — train loop, eval MSE + cosine sim
- [ ] `src/baselines/` — train ConvLSTM, Action-CNN, DreamerV3-style song song
- [ ] Offline evaluation trên held-out val set

### Phase 4 — CEM Planning + Eval (tuần 4)
- [ ] `src/cem_planner.py` — CEM: sample action sequences → AC Predictor → pick best
- [ ] `src/inference_loop.py` — real-time: capture → encode → plan → command
- [ ] Goal-conditioned test trong nhà
- [ ] So sánh vs baselines, viết report

## V-JEPA 2.1 Notes

- Weights: download thủ công từ `https://dl.fbaipublicfiles.com/vjepa2/`
- HuggingFace (dev/Colab): `facebook/vjepa2-vitl-fpc64-256`
- Model: ViT-L, fpc64 (64 frames per clip), input 256px
- Output: spatial tokens `(T/2, H/16, W/16, 1024)` → mean pool → `(1024,)`
- KHÔNG bao giờ backprop qua encoder — frozen hoàn toàn
